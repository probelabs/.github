# Welcome to ProbeLabs!

We build intelligent tools for developers that provide deep insights into their codebases. Our products focus on code search, AI-powered code review, and traffic replay to help teams build better software, faster.

## Our Core Projects

Here are some of our key open-source projects:

*   Probe: A semantic code search engine that understands your code's meaning, not just keywords.
*   Visor: An AI-powered code review assistant that helps you catch bugs and improve code quality automatically.
*   GoReplay: An open-source tool for capturing and replaying live HTTP traffic into a test environment.
*   Logoscope: An AI-powered log analysis tool to help you find the needle in the haystack.
*   Maid: A linter for Mermaid diagrams to keep your documentation clean and consistent.
